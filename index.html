About this weather app

How the data is fetched and how the UI updates.

1) Weather data

This app uses the OpenWeatherMap 5 day / 3 hour forecast API. When you search a city (or use your location), it fetches JSON and then renders: current (first item), hourly (next 24 hours), and a 5-day snapshot (midday).

2) Search + recent cities

Your searches are saved in localStorage (up to 5 cities). The dropdown appears only after your first search.

3) Units + alerts

The °C/°F toggle changes only today’s temperature. If the temperature is above 40°C, the app shows a custom toast warning.

4) Dynamic theme

Background tint + effects change based on weather (clear/clouds/rain/thunder/snow/night).

Tip: If you open the file directly (file://), the browser can block API calls. Use Live Server / localhost.
